About Me

저는 에스트라의 피부과학 철학에 깊이 공감하며, 더마코스메틱 콘텐츠 제작자로서의 성장을 꿈꾸고 있습니다. 사용자와 제품을 잇는 감성 콘텐츠를 기획하고, 피부에 진심인 브랜드 메시지를 시각적으로 표현하는 것을 좋아합니다.

강점

  • 프론트엔드 개발 능력 (HTML, CSS, JS)
  • 감성 콘텐츠 기획 및 디자인 경험
  • 더마 브랜드에 대한 관심과 이해도

기술 스택

  • HTML / CSS / JavaScript
  • Figma, Canva, 미리캔버스
  • GitHub / VS Code 활용

비전

  • 브랜드에 대한 애정을 시각화하는 디지털 크리에이터
  • 소비자의 피부 고민에 공감하는 콘텐츠 제작
  • 에스트라의 가치를 알리는 브랜드 앰버서더
  • 자작곡: 에스트라

에스트라 스킨케어 제품을 써보고 느낀 감정과 A마스터에 대한 마음가짐을 AI작곡을 통해 표현했습니다.
AI툴을 활용한 작사 작곡 경험이 풍부합니다.
이미지를 클릭하여 자작곡을 재생해보세요.

  • 다빈치 리졸브

다빈치 리졸브를 활용하여 영상 편집 및 색 보정이 가능합니다.
편집 툴을 활용해 에스트라 제품 리뷰, 에스트라와 함께하는 관리 브이로그를 촬영하고 싶습니다.

  • 노션

노션을 자유자재로 활용합니다.
노션 프로필, 노션 홈페이지, 계획표 등을 나만의 스타일로 만들 수 있고,
노션을 활용한 경험이 풍부합니다.

  • 코딩

GSITM 부트캠프를 통해 코딩기술을 배웠습니다.
특히 프론트엔드에 능하며 웹페이지 디자인, 구상능력을 가지고 있습니다.
실제로 웹페이지를 기획하고 실현하는 프로젝트를 수행하였습니다.

  • 블로그 운영

작년부터 현재까지 활발히 블로그를 운영하고 있습니다.
카페 리뷰, 대외활동 후기, 감성글(시, 소설)을 창작하여
감성을 저격하고, 공감을 부르는 콘텐츠를 제작합니다.
